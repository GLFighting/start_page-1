项目名称 BYD startpage


**网站起始页，包含自定义引擎搜索，日历时间小组件，备忘录以及todolist和自定义壁纸，让你快速开启网络链接,自由页面管理，轻松畅游网络世界**

## 主要技术栈：
### 前端
- vite2
- vue3
- animate
- less
- typescript
- javascript + html + css
- axios
### 后端
- springboot
### UI设计
- figma

## 功能描述
1. 备忘录提醒
2. 时间提醒
3. todolist待办事项
4. 日历
5. 自定义壁纸
6. 自定义搜索引擎
